# clist [![CircleCI](https://circleci.com/gh/Qix-/clist.svg?style=svg)](https://circleci.com/gh/Qix-/clist)

The missing C lists library.

## Building

```console
$ mkdir build && cd build
$ cmake .. -DBUILD_TESTING=On -DCMAKE_BUILD_TYPE=Debug    # or =Release
$ cmake --build .
```
